jdk安装

  • 解压 tar -zxvf jdk-8u191-linux-x64.tar.gz

环境变量配置

  • 修改/etc/profile
  • source /etc/profile 修改生效
  • 创建软连接 方便版本更换和学习使用
  • 修改.bashrc 文件
  • 这些环境变量的权限控制到用户级别,这里是针对某一个
    特定的用户，如果你需要给某个用户权限使用这些环境变量，你只需要修改其个人用户主目
    录下的.bashrc 文件
  • source ~/.bashrc 修改生效

scala安装

发表于 2019-01-25 | 分类于 scala

scala安装

  • 解压
    tar -zxvf scala-2.11.8.tgz
  • 建立链接
    ln -s scala-2.11.8 scala

    环境变量配置

    • 修改.bashrc 文件
      这些环境变量的权限控制到用户级别,这里是针对某一个 特定的用户，如果你需要给某个用户权限使用这些环境变量，你只需要修改其个人用户主目 录下的.bashrc 文件
  • 查看scala版本
    scala -version
